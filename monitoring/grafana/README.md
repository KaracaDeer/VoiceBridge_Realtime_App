# Grafana Dashboard Configuration

This directory contains Grafana configuration files for monitoring.

## Dashboard

The main dashboard configuration has been moved to `docs/monitoring/grafana-dashboard.json` to keep the repository clean.

## Usage

1. Copy the dashboard from `docs/monitoring/grafana-dashboard.json`
2. Import it into your Grafana instance
3. Configure data sources as needed

## Configuration

- Dashboard ID: VoiceBridge API Monitoring
- Tags: voicebridge, api, monitoring
- Style: dark
- Timezone: browser
